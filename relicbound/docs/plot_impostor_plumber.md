# 剧情设定：冒充水管工 (Impostor Plumber)

## 一句话概念
两个冒牌水管工被派去修复一座废弃像素公寓的管道系统，但他们根本不会修水管——玩家必须在"假装很专业"和"真的把事情搞砸"之间走钢丝。

## 世界观定位
- 发生在 Relicbound 宇宙的支线关卡，独立于主线 ARPG 进程，但可与主线道具产生联动。
- 基调：轻松喜剧，带一点存在主义荒诞。两个角色知道自己很菜，但他们更怕失业。
- 美术风格：与主线一致的像素风，但色调偏暖（黄铜色管道、米白瓷砖、昏黄走廊灯），营造一种"老旧公寓的温馨感"。

## 角色设定

### 角色 A：老金 (Lao Jin)
- 年龄：40出头，自称"干了二十年水管工"，但实际上从未通过任何资格考试。
- 性格：表面老练、爱摆架子，喜欢用行话唬人（"你这属于主立管回水不畅，得从总阀查起"），内心慌得一比。
- 小动作：紧张时会拧扳手，即使面前根本没有螺丝。
- 动机：想攒够钱开一家真正的咖啡店（所以他对咖啡机特别执着，每次路过都会停下来摸两下）。
- 玩家感受锚点：他的"装懂"会让玩家笑，但他偶尔流露出的真诚（比如担心自己骗不了别人）会让玩家对他产生同情。

### 角色 B：小阿 (Xiao A)
- 年龄：20出头，老金的侄子/远房亲戚，跟着来"学手艺"。
- 性格：天真、好奇心重、什么都敢问。恰恰是他那些"外行问题"经常戳穿老金的谎言。
- 小动作：喜欢用手机搜"如何看起来像个专业人士"，然后在老金面前假装很有把握地复述搜索结果。
- 动机：其实他想当艺术家，但被老金忽悠着签了两年合同。他在偷偷画公寓里遇到的每只猫。
- 玩家感受锚点：他是玩家的"良心"——当老金要做出荒谬决定时，小阿会发出质疑。但玩家控制的是两人切换，所以最终选择权在玩家手里。

## 核心玩法循环

### 1. 接单
- 公寓里的住户（NPC）打电话来报修：漏水、堵塞、热水器不工作、"好像有东西在管道里唱歌"。
- 老金会在电话里过度承诺："没问题，半小时搞定！"（实际上他连故障原因都不知道。）

### 2. 诊断（假装环节）
- 玩家进入场景后，需要先"观察"问题。但观察工具是有限的——只有一个手电筒、一个内窥镜摄像头，和一个老金从二手市场淘来的"多功能管道检测仪"（其实是个坏掉的收音机）。
- 玩家需要通过有限的线索拼凑故障原因，但如果线索不足，就得靠"猜"。
- 猜对的概率取决于玩家收集到的线索数量。全猜 = 高风险高回报。

### 3. 维修（执行环节）
- 一旦确定了"治疗方案"，玩家切换到对应的小游戏模式：
  - **管道疏通**：类似节奏游戏，需要在正确时机按下按键来旋转管道接头。
  - **焊接修补**：控制焊枪温度，太高会烧穿管道，太低粘不住。
  - **更换零件**：从一堆零件中选出正确的一个（考验玩家的观察力）。
- 每个小游戏都有"完美操作"和"勉强过关"两个档位。完美操作赢得住户好评，勉强过关则住户半信半疑。

### 4. 伪装成功 / 露馅
- 如果维修成功：住户付钱，可能留下小费或线索道具。老金会得意忘形地说一些更离谱的谎话。
- 如果维修失败：管道喷 water、墙壁渗水、或者出现更严重的问题。住户会打电话投诉——然后触发一个"补救任务"，玩家得在住户找物业之前再次潜入修复。
- **关键设计**：失败不是 game over，而是另一种剧情的开始。就像现实中很多糟糕的水管工只会越修越糟，但故事还在继续。

## 关卡结构（示例：第一关）

### 场景：梧桐公寓 3 栋 402 室
- 住户：一位独居的老奶奶，养了三条金鱼。
- 报修内容："小伙子，我厨房的水龙头一直在滴水，还有……你们说修水管的，能不能帮我看看鱼缸？最近鱼不太爱动。"
- 玩家同时处理两个问题：水龙头（常规维修小游戏）和鱼缸（意外发现的连锁问题——原来是楼上漏水导致水质变化）。
- 结局分支：
  - 如果玩家顺便修好了鱼缸加热棒：老奶奶感激涕零，给了一个旧怀表（可出售或作为收藏品）。
  - 如果玩家只修了水龙头：老奶奶有点失望，但没说什么。
  - 如果两个都搞砸了：鱼翻了白眼，老奶奶打电话给她儿子——她儿子是物业经理，接下来会派一个"检查员"来。

## 叙事主题
- **冒充者的困境**：老金和小阿都在扮演一个他们不够格的角色。这不是讽刺，而是共情——每个人都在某种程度上"冒充"着自己生活中的某个身份。
- **失败的尊严**：游戏不惩罚失败，而是让失败变成故事的一部分。你修不好水管没关系，你至少尝试了，而且你还得收拾残局。
- **微小善意**：最好的结局往往不是完美的维修，而是玩家多看了一眼鱼缸、多问了一句"您最近睡得好吗"。

## 与主线的潜在联动
- 老金怀里的"多功能检测仪"（坏收音机）在主线关卡中可以作为一个彩蛋出现——当玩家靠近它时，会随机播放一段来自 Relicbound 宇宙其他区域的"信号"，暗示这个世界比表面上更大。
- 小阿画的公寓猫咪素描集可以作为收藏品出现在主线中，每张素描解锁一段关于这栋公寓过去的碎片化叙事。

## 参考作品 & 调性对标
- 《Plumber Simulator》（Steam 上的搞笑模拟游戏）的荒诞感。
- 《Untitled Goose Game》的"故意捣乱"乐趣。
- 《星露谷物语》中老姜那种"嘴硬心软"的角色弧光。
- 对白风格参考：《瑞克和莫蒂》的快节奏互怼 + 《马男波杰克》的存在主义自嘲（当然像素版会更温和）。
