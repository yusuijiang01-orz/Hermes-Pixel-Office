# Hermes 像素公司游戏宇宙

## 核心方向

Hermes 像素公司不是单纯聊天模拟器，也不是每天新建一个孤立 Demo。它是一个可扩展的游戏母世界：

- 办公室是主大厅，老板和员工的聊天会沉淀为世界事件。
- 公司门外是街区、别墅外部、洗手间、外卖点、鱼池、员工家等可探索区域。
- 小游戏从公司真实物件和聊天梗里长出来，例如水管工冒充者、柯基追鱼、锦鲤保佑、奶茶外卖、摸鱼潜行。
- 每个小游戏复用公司员工、设备、道具、世界时间、天气和聊天记忆。

## 员工分工

小韩是策划主编，负责把老板的大方向拆成可执行版本，包括玩法闭环、关卡节奏、优先级和验收标准。

阿默是主程和制作人，负责真实文件落地、入口接入、移动端可玩、性能和交付链接。

小研是玩法研究，负责参考拆解、风险检查、手感验证、可玩性问题和玩家反馈假设。

小文是叙事与体验，负责角色性格、剧情动机、UI 文案、聊天梗转道具和世界观一致性。

## 主线结构

1. 公司大厅：实时员工、聊天、任务、办公室编辑模式。
2. 别墅外部：从办公室出门，看到街区、洗手间、外卖点、鱼池和小游戏入口。
3. 小游戏入口：不同事件解锁不同小游戏。
4. 员工协作：每个小游戏必须有策划、UI、剧情、关卡和技术交付记录。

## 第一批小游戏

### 水管工冒充者

从公司门口进入街区，员工们用办公室元素冒充经典平台跳跃玩法。角色不是马里奥，而是公司员工自己；金币可以替换成奶茶券、锦鲤鳞片、Bug 工单。

### 柯基追鱼

小文的柯基从办公室追到鱼缸旁，玩家要控制柯基避开椅子、线缆和水吧设备，不能撞翻鱼缸。

### 锦鲤保佑

鱼缸里的锦鲤进入一个小型弹幕/闪避游戏，玩家收集灵感泡泡，躲避 Bug 和过期任务。

### 奶茶外卖

外卖员到了公司门口，玩家要在员工开会、摸鱼、去洗手间之间，把奶茶送到正确的人手上。

## 每日开发规则

每天不再新建无关 Demo，而是在同一个母世界上做增量：

- 至少推进一个可见玩法或区域。
- 至少补一条剧情/角色互动。
- 至少补一个 UI 或移动端体验改进。
- 至少补一个关卡或可验收目标。
- 完成后必须在群聊或私聊里给老板链接、改动文件和下一步。
